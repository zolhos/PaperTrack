# Base de Conhecimento - PaperTrack

Esta pasta contém a base de conhecimento e informações úteis para o desenvolvimento do **PaperTrack**.

## Estrutura de Diretórios
- `/citation-rules`: Regras e especificações de formatação bibliográfica (ABNT, APA, etc).
- `/spotify-api`: Documentos de referência sobre endpoints, esquemas de JSON e fluxos de autenticação do Spotify.
- `/research-notes`: Notas de pesquisa científica e história oral acadêmica sobre o valor de fontes orais.
